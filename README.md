## Установка и запуск

```bash
# Step 1: Clone the repository
git clone 

# Step 2: Navigate to the project directory
cd 

# Step 3: Running container
docker-compose up -d

# Step 4: Enable execution
chmod +x 

# Step 5: Run script


# Step 6: Access the application
Open browser and go to http://localhost:8080
```
